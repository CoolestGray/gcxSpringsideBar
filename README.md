# gcxSpringsideBar 
## gcxSpringsideBar  - sidebar cool easy
## 开发
```bash
    # 克隆项目
    git clone  https://github.com/Lensgcx/gcxSpringsideBar.git
    # 安装依赖
    cnpm install
```
## 使用示例
```bash
   $( '#tabpanel' ).gcxSpringsideBar(
   		defaults = {
   			size:'400'
  			skin: 'greenleaf'     
   		}
   );
```
